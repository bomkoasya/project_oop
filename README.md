# Personal Finance Manager 

**Менеджер особистих фінансів з прогнозом** – це зручний інструмент для трекінгу ваших доходів і витрат, управління бюджетом та аналізу фінансів за допомогою графіків та прогнозів.

---

## Основні можливості

- **Трекінг витрат та доходів**  
  Ведіть облік усіх фінансових операцій з деталізацією по категоріях.

- **Категорії та бюджети**  
  Створюйте категорії витрат (їжа, транспорт, розваги тощо) та встановлюйте щомісячні бюджети.

- **Графіки та аналітика**  
  Візуалізація доходів і витрат за категоріями, періодами та балансом.

- **Прогноз витрат**  
  На основі історії транзакцій система робить простий прогноз наступних витрат, щоб ви могли планувати фінанси заздалегідь.
